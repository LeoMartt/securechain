# SecureChain Audit

Projeto prático da disciplina Segurança de Sistemas com Blockchain, Criptografia e Auditoria de Eventos.

## Ambiente

- Sistema Operacional: Debian 13 em Máquina Virtual
- Linguagem: Python 3
- Scripts: Bash Script
- Versionamento: Git + GitHub

## Estrutura inicial

- blockchain/: implementação da blockchain e arquivo chain.json
- auditoria/: coleta de dados do sistema operacional
- backup/: script de backup seguro
- logs/: registros locais
- documentos/: arquivos monitorados
- usuarios/: dados de autenticação com senhas em hash

## Usuários do sistema

- administrador: acesso total
- analista: leitura e execução
- visitante: leitura dos relatórios
